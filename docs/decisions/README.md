# Architectural Decision Records

This repository records architectural decisions in `docs/decisions` using an
adapted MADR 4.0.0 template.

## Workflow

1. Pick the category that best matches the decision.
2. Copy `docs/decisions/adr-template.md` into that category.
3. Rename it to the date (YYYY-MM-DD) plus a short dashed title.
4. Replace placeholders, remove sections you do not need, and keep the ADR immutable after finalized except for clarifications.
