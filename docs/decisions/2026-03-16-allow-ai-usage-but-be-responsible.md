# Decision making and implementation responsiblity should remain to humans, not AI

## Context and Problem Statement

AI agentic coding is hot, but not sure how much AI is okay for this project.

## Considered Options

- No AI at all
  - Write everything by hand, don't accept PRs written with AI
- Some AI
  - It is fine to use AI, but make sure the decisions and responsibility is on humans.
- All AI
  - Full-on vibe coding

## Decision Outcome

Chosen option: "Some AI", part of this project is for me to enjoy and learn, and I don't want to just have some super vibe-coded product.
